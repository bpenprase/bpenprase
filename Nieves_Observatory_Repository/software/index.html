NSUT Software Overview

Astronomical Image Processing Suite

Based on your uploaded software overview guide, this section describes the four core programs used for Nieves Soka University Telescope data analysis. Upload Python scripts to content/software and they will appear in the file listing below.

Typical End-To-End Workflow

  1. ExposeCheck: identify saturated FITS frames and remove overexposed images.
  2. FITS Image Combiner: create master calibrations and aligned stacked science frames.
  3. PhotoCalib: perform robust differential photometry across the time-series stack.
  4. Differential Photometry Plotter: generate publication-ready light curves and diagnostics.

ExposeCheck

Scans FITS directories for saturation using target, center, or full-frame modes, then exports CSV diagnostics.

Primary file: exposecheck.py

FITS Image Combiner

Combines and aligns FITS frames using average, median, sigma-clip, or brightest-pixel rejection methods.

Primary file: fits-image-combiner.py

PhotoCalib

Computes differential photometry from WCS-solved images with target/reference tracking and diagnostics.

Primary file: photcalib_robust_fixed.py

Differential Photometry Plotter

Converts photometry CSV output into two-page, publication-quality multi-panel light curve reports.

Primary file: plot_differential_photometry.py
