# Future-Proofing the University - Website

A beautiful, responsive website promoting Prof. Bryan Penprase's book "Future-Proofing the University: Innovations in Learning Technology and Finances in the Age of AI"

## Features

- **Responsive Design**: Mobile-friendly layout that works on all devices
- **Beautiful UI**: Modern gradient design with smooth animations and transitions
- **Interactive Chapters**: Click to expand/collapse chapter abstracts
- **Author Information**: Detailed bio and information about Prof. Penprase
- **Previous Works**: Showcase of author's previous publications
- **GitHub Pages Ready**: Deploy directly from the repository

## Setup Instructions

### 1. Add Your Book Cover

Replace the `book-cover.jpg` file with your actual book cover image:
- Recommended size: 300px width (scales responsively)
- Formats: JPG, PNG, or WebP
- The image will display in the hero section

### 2. Deploy on GitHub Pages

1. Ensure the repository is public (or Pages is enabled on private repos)
2. Go to repository Settings → Pages
3. Set source to: `main` branch → `/FutureProof_Universities` folder
4. Your site will be available at: `https://bpenprase.github.io/FutureProof_Universities/`

### 3. Customize (Optional)

Edit the following files to customize:
- `index.html`: Update content, titles, or structure
- `styles.css`: Modify colors, fonts, or layout
- `script.js`: Add or modify chapter data

#### To change colors in CSS:
Open `styles.css` and modify the `:root` variables:
```css
:root {
    --primary-color: #1e3a8a;      /* Blue */
    --secondary-color: #0f766e;    /* Teal */
    --accent-color: #dc2626;       /* Red */
    /* ... */
}
```
